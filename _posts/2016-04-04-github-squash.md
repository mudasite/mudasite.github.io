squash できるようになったときいてちょっとためしてみる

- ユーザアカウント側で fork
- branch 作成
- Markdown 編集
- コミット
- PullRequest 作成
- organization 側でマージ

ってやったあと、ユーザアカウント側の fork を Web 上で最新化(pull)する方法がわからない。。
